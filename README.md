# serverless-koa-micro

基于 serverless + koa 实现 Node 微服务

## download

> git clone https://github.com/hubvue/serverless-koa-micro.git

## start

### install serverless

首先在全局安装 serverless

> npm install serverless

or

> npm install serverless -g

### run

> npm run dev

or

> yarn dev

### test

> npm run test

or

> yarn test

### deploy

需要按照 serverless 官方的步骤获取到 AWS 的 key

> npm run deploy

or

> yarn deploy
